## 🔨 How to run the project

```bash
# Clone this repository
$ git clone https://github.com/tl-lucasestevam/dash-go

# Access the project folder
$ cd dash-go


# Install the dependencies
$ yarn

# Run the application in development mode
$ yarn dev

```
